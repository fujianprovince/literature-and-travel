## 《松柏集》  
* 一  
家院（陈郑逸）  
房前小池古树，清风徐徐。  
屋后水田秀山，翠色茫茫。  
* 二  
赞龙翔水（陈郑逸）  
龙翔自然水，城里差几何。  
鸟鸣山涧取，泉涌溪水得。  
清透如翡翠，甘洌赛珍蜜。  
渴饮沁心脾，累抿怡神气。  
* 三  
晨学（陈郑逸）  
清晨五更寒，书声琅琅来。  
若恋被窝里，怎得见初阳？  
* 四  
作业（周政康）  
一支笔，一本书。开着台灯不吭声。看看天，看看书。想想明天要上学，心里一阵凉。  
* 五  
江湖·梦（周政康）  
几人去，几人回 ，风流一生，痛苦一生，好似一场梦。  
* 六  
坝（陈郑逸）  
建坝堵决口，日月累砂石。  
若懈半分毫，顷刻即塌陷。  
人生亦如此，败易成功难。  
* 七  
琴筝（陈郑逸）  
木琴经久练，弹来悠远音。  
楠筝新华丽，奏出不润声。  
故若要成就，必要有实践。  
* 八  
成学（陈郑逸）  
贫生贪早诵，持续多艰难。  
只为立家业，重振沦落家。  
富子恋游玩，度岁可逍遥。  
不思持族事，大败辉煌户。  
所谓要学成，定需久历难。  
此可比投资，今苦将来甘。  
* 九  
鹤与翁(陈郑逸)  
墨草簇水岸，池中微翻滚。  
三鹤饥伫旁，欲食鱼饱腹。  
首发第一鹤，惫懒且惰想。  
伸长脖巴望，欲吃怎吃得。  
复来第二鹤，有力但倦思。  
展翅扑进水，鱼惊瞬无存。  
末出第三鹤，乐谋不望劳。  
尽策千百计，无一实有行。  
三鹤正无食，渔翁笑吟来。  
饵钩入水晌，扑腾上钩来。  
莫赶笑三鹤，且先省自己。  
平日学习法，似鹤还如翁？  
* 十  
天地间(陈郑逸)  
畏望地坪通七窍，难览天篷布四边。  
天拦地护防灾害，凡人一点在其间。  
* 十一  
父母的爱(陈郑逸)  
父亲的爱，  
是积极的鞭策，是明智的向导;  
母亲的爱，  
是清凉的风，是遮雨的伞。  
父爱如山，抵挡风雨侵害;  
母爱似海，包容星罗万象。  
一生关护，都为我成长;  
一世辛劳，全送您变老。  
待我长大，您已白发，  
再欲报答，将复何及？  
* 十二  
食与颂(陈郑逸)  
加一毫味少鲜，油不亮色缺美。  
武林试味难得，汉子欲食不到。  
要将此食与颂，送与白衣英豪！  
* 十三  
战乱(陈郑逸)  
一日复一日，清风化硝烟。  
晴空为混沌，鸟鸣改炮言。  
丰景已不在，人烟更不存。  
要拿天问罪，妖来何不闻？  
* 十四  
见秋(陈郑逸)  
独遇风，风凄凉，感人凉，不禁伤。怅然若失皆不知，切莫凭栏望，若望风舞叶，更欲伤。  
* 十五  
阳光一直都在（陈郑逸）  
  日升五更，家窗透明。每朝睁眼，即是光明。  
  精神焕发，更衣出户。天前二三飞鸟告晓，屋后四五走鸡报晴。天自映地，地亮衬天，众山恭排日下，群林谨守光前，阳光漫天下，朝气腾四边。  
  路前润润水潭蓄翠，树下圈圈水纹亮边。不时水珠溅跃，浴于光间，原来蜻蜓闲步。岸旁树干有光点游走，头上叶丛见晴光历历。明棕墨绿，暗褐艳翠，交融缤纷，灵动似活。更有乱草铺石间向阳摇摆，迷花立其中牵光共舞，蜂虫同欢唱，蚁蛛也探头。一片炊烟飘上头，人间闹市亦纷华。  
  但到黄昏日睡后，便不复阳光、不复生机？非也，人城民坊，所谓“阳光”更甚。华灯初上，火树银花。车水马龙，摩肩接踵。人前百车走，车侧万人行，皆为国富民强奔波生计。阳光一直都在，吾民族有望矣。  
2020年10月21日深夜于福州  
2020年10月28日改动几字（原为作业，今录入，“华灯初上，火树银花”为老师所添而作者采纳者）  
* 十六  
乡愁（陈郑逸）  
烟烟天哭雨，昏昏人欲愁。  
鱼酒是丰盈，哪有味更浓。  
低低流水暗，映鸟时低头。  
一声家书来，顿醒池边洲。  
* 十七  
散学有感（陈郑逸）  
星南月北亮晃晃，车漫人移影绰绰。  
要见鸿鹄真效果，像欲惑我我不惑。  
* 十八  
草（陈郑逸）  
何惧北风刮东西，春来根生绿万顷。  
自强不息风骨在，天下何难不惧君！  
（注：陈郑逸一幅仿国画《顽草图》，题词即此）  
* 十九  
夜深独台（陈郑逸）  
漆漆月风高，伫久不觉寒。城灯缥缈，欲张眼，不辨是雾或泪帘。  
飕飕风骨削，要走英才气。此挫何尔？欲凌空，不论风暴与山险！  
（注：此篇为一英语考试较难失利后作）  
* 二十  
家烛颤（陈郑逸）  
眼隙看处，灯红倒数，恰恰危急时刻，滚滚绿闪埋没，人骑无熟影，恍惚东西却无人！玻瓷渣声落心间，只欲缠头掩泪溅，恨把心抓要人现。吾誓不容瑕疵现，卒也浮现学江海。昔日家欢逝于何？！  
（注：此篇为父母因子学之事等一吵后作）  
* 二十一  
纵江海（陈郑逸）  
人生总有失意处，成功难得几轮回。  
漂浮不定学江海，誓倚巨浪腾空飞。  
* 二十二  
学疲有感（陈郑逸）  
人身非铁铸，劈头打磨折。  
似水走流年，经久方为真。  
* 二十三  
冬至（陈郑逸）  
一点白烟几处朦胧，气腾出锅，暖手出白裹。  
一点灯火几面映红，顺滑入胃，团圆是隽永。  
* 二十四  
氏名意（陈郑逸）  
陈门松柏凝正气，名邸书香使逸情。  
* 二十五  
海王赋（陈郑逸）  
千古涟漪海王地。海岱楼高，下瞰众鱼。  
水浸碧天天似水。美丽宫阙鱼间世。  
白浪茫茫与海连，沙丘浩浩四无边。  
海鱼群群囍嬉戏，生机勃勃主为谁？  
原是海王黄涵宇，大寿十四庆海晏。  
* 二十六  
自励铭（陈郑逸）  
段五十五屈行列，泽神能五势难追。  
寄人篱下方恨少，卓越者还未闻谁。  
落雨淅沥趁未尽，切莫枉负少年岁！  
（原“六十五”于2021年11月21日由作者改为“五十五”，原“十四”于2022年3月23日由作者改为“能五”）  
* 二十七  
注定不平凡的雨天(陈郑逸）  
//  
当连日的淫雨霏霏，  
打压着燥热的暑气，  
是茫茫世界被泪水染遍，  
注定不平凡的雨天。  
//  
当午后晴空被黑云覆盖，  
当蚊虫也呜咽着离开;  
当空气不再快活，  
当自然的声音也充斥着不安，  
注定不平凡的雨天。  
//  
当晾晒的衣物带上沉重的累赘，  
当来往的街道遍布污浊的泥水;  
车辆驱离，人群四散，  
屋檐，伞，雨衣，  
逃避，慌乱，心烦，  
注定不平凡的雨天。  
//  
当雨摇曳着世界，  
当泪在揪紧人心;  
当雨清洗着世界，  
当泪也洗涤心灵。  
将自己寄人篱下，  
犹如将生命敞开向暴风雨;  
趁雨水还未极其悲愤，  
趁狂风还未极力怒吼，  
是否能通向黎明……  
注定不平凡的雨天。  
//  
暴风雨愈发猛烈，  
黎明将近却无法触碰;  
容不得我半点等待！  
请让风冲刷我的腐朽，  
请让雨磨砺我的意志，  
让黎明早早到来吧！  
让血泪染遍世界！  
注定不平凡的雨天！  
//  
2021年8月5日 暴雨

* 四十五  
无题（陈郑逸）  
  吵架真是世上最可恨的东西，甚至于它都不怎么能配上“东西”这样的称呼。它复杂，往往难以互相说服，旁观者也难以判断对错;它一瞬间伤害极大，好比矿井爆炸的强度，没有人劝阻的话这往往引起更可怕的后果;它难以短期愈合，不管是心理上还是物质上的创伤。这大概由于人的思维过于复杂，因为事物总有两面性，相比其他生物，思维的有利影响多得多，不利影响却更完全不可避免了。因此，人们吵架的出发点往往有共性，那是随意去除不了的。问题就在这，我为此彻夜难眠，多少人彻夜难眠，我们失去的都被该死的吵架夺去了。

* 五十一  
一纸酒精（陈郑逸）  
风儿里自顾着打量玻璃罐，  
只为让酒精浸湿里头的纸。  
靠近着点燃。  
阳光掩盖淡蓝的、飘忽不定的火苗，  
只有隐约的、扭曲的空气，  
尽力地、也是微弱地宣告燃烧在进行。  
怎能耐得住这样平凡的表现，  
于是旋上玻璃罐的盖。  
火苗不知何时去，  
酒精作雾满罐墙。  
火灭了。  
——呵，  
不过如此呀，  
旋下玻璃罐的盖;  
还能怎样啊，  
再靠近着点燃。  
嘭——！  
火光瞬起，  
腾蔓四边。  
从罐里吐出的，  
是翻滚跳跃的火圈;  
从火圈绽放的，  
是阳光盖不住的光和热！  
爆炸——！  
——持久的静寂。  
远离着点燃。  
——噗嗤。  
酒精将尽，  
余热寥寥。  
——火灭了，  
但留一纸一成不变的洁白！
